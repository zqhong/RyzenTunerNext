 系统建议
根据测试结果，我们建议：
- 省电模式 设置为 15W，此时能效比极佳，温度极低。
- 性能模式 设置为 30W - 35W 为甜点区间，超过 45W 后性能提升不足 5%，但温度急剧升高且触发了温度墙 (95℃)。
